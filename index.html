Spacejam example

This is a basic 3d rendering engine based on the canvas-element written in Javascript. Every single pixel is calculated on the CPU every frame. Think about that. I marvel at it every time I do.

How cool is it that software (and my own code for that sake) can do this?! :D

I made this to improve my understanding of 3d graphics. It has been a really good exercise.

Of course, it should not be used in production. The GPU would be a gazillion times faster.

You can check out the full source code at Github.
